# 🍽️ Food Recommendation System

## 📌 Project Overview

The **Food Recommendation System** is a simple Artificial Intelligence-based project that suggests food items based on user preferences. The system uses basic **pattern matching and filtering logic** to recommend food according to inputs such as food type and taste.

This project is developed as part of an **AI Internship (Project 3: Recommendation Logic)** to demonstrate the fundamentals of recommendation systems.

---

## 🎯 Objectives

* To understand the core concept of recommendation systems
* To implement user preference-based filtering
* To apply basic AI logic using pattern matching
* To build a simple and interactive Python application

---

## 🧠 How It Works

1. The system takes user input:

   * Food Type (Veg / Non-Veg)
   * Taste Preference (Spicy / Sweet)

2. It compares the input with a predefined dataset of food items.

3. Based on matching conditions, it recommends relevant food options.
